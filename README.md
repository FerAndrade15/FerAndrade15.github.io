# My Engineering Portfolio

A small static portfolio website powered by **Bootstrap + PyScript**.  
It displays a list of projects and runs interactive Python tools directly in the browser.

## 🚀 Run Locally

1. Open VS Code in this folder.
2. Right-click `index.html` → **Open with Live Server**.
3. The site will open at [http://127.0.0.1:5500/Portfolio/](http://127.0.0.1:5500/Portfolio/).

If you don't use Live Server, you can also run:
```bash
python -m http.server
